# Claude Skills

A personal collection of Claude Code skills for scientific research, data science, analytics, and teaching.

## Skills

| Skill | Description |
|-------|-------------|
| `jurisprudence.skill` | Maintains a per-project jurisprudence log: implicit rules, conventions, clarified assumptions, and decision precedents |
| `cowork-safety-check.skill` | Checks Cowork-style tasks (file access, browser/MCPs, scheduled tasks, computer use) against Anthropic's 10 official Cowork safety rules and surfaces warnings before risky actions |
